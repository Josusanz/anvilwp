## [1.0.0] - 2026-02-27

### Added
- Initial release of AnvilWP
- Core theme generation functionality
- Dark neon aesthetic based on Cruip/Neon quality standards
- Claude Code integration for developers
- Built-in SEO features (Open Graph, Twitter Cards, canonical URLs)
- Native contact form with REST API endpoint
- Schema.org JSON-LD structured data
- Cookie consent banner (GDPR compliant)
- Performance optimizations (remove WordPress bloat)
- Responsive design with 900px and 600px breakpoints
- Pre-designed patterns:
  - Hero with grid background and glow effects
  - Services/Features with circular gradient icons
  - Stats with large gradient numbers
  - Testimonials with avatars
  - Team section with professional cards
  - CTA call-to-action section
  - Contact form with validation
- Complete documentation:
  - Quick Start Guide
  - Claude Code Guide with prompt templates
  - Design Patterns with quality standards
  - Contributing guidelines
- GPL-2.0 License

### Technical Details
- WordPress 6.4+ with Full Site Editing (FSE)
- Native Gutenberg blocks only
- theme.json configuration with useRootPaddingAwareAlignments
- CSS Custom Properties for easy theming
- Vanilla JavaScript (no jQuery dependency)
- Plus Jakarta Sans + Inter typography
- SVG icons with radial gradients
- Smooth scroll animations
